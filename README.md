FrontInVale 2017
May the force be with the front!

### Setup & install dependencies

```
$ git clone git@github.com:frontinvale/frontinvale.github.io.git
$ cd froninvale.github.io
$ git checkout dev
$ npm install && bower install
```

### Server

```
$ gulp serve
```

### Access:

```
http://localhost:9000
```

### Automatic deploy

```
$ gulp deploy
```
